# ChatGPT Work Billing Transparency

A public, source-backed consumer transparency project documenting the relationship between ChatGPT Work, Codex usage limits, shared credits and Auto top-up.

## Core documented facts

- OpenAI states that Codex, ChatGPT Work and supported agentic features can draw from the same usage and credit pool.
- OpenAI states that Work uses the same pricing, credits and limits as Codex.
- Shared credits may be consumed across supported features after included usage is exhausted.
- If a user deliberately enables Auto top-up, the shared balance may trigger a purchase.
- Credits generally expire after 12 months and are generally non-refundable, subject to law.
- Image generation may consume included agentic limits roughly 3–5x faster on average.

## Accuracy policy

Every material statement is classified as documented fact, reasonable inference or not established. The project does not allege criminal conduct or claim evidence of corporate intent.
