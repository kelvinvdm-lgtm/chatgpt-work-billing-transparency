# ChatGPT Paid Usage Pool Transparency

A public, source-backed consumer transparency project focused on one central issue: multiple paid agentic products can compete for the same included usage allowance and shared credit balance.

## Central documented facts

- OpenAI states that Codex, ChatGPT Work, ChatGPT for Excel and Workspace Agents can draw from the same agentic usage and credit pool when available.
- A subscriber's included usage is consumed before purchased credits.
- Credits purchased through Codex may be consumed by other supported products.
- If a user deliberately enables Auto top-up, activity across the shared balance may contribute to a purchase.
- OpenAI says individual-service content may be used to improve models unless the user opts out.
- OpenAI now publishes token-based Codex credit rates for most customers, with a small Enterprise subset still on the legacy per-message card.
- For ChatGPT Business, OpenAI explicitly includes PowerPoint in the shared agentic pool; charging begins 6 August 2026.

Last verified against official OpenAI sources: 12 July 2026.

## Consumer position

Users do not need to demand wages or ownership of the model to expect fair value. When paying customers help finance the service and may also contribute useful real-world data, new products should not quietly dilute the same paid work allowance. Add capacity, provide separate pools, or disclose the trade-off unmistakably.
